# CAM Cleanup — CapabilityAccessManager スリム化ツール

Windows 11 の一部環境で発生する **CapabilityAccessManager.db-wal の異常肥大化**によって失われた
Cドライブの空き容量を、**ダブルクリック一発**で取り戻すためのバッチファイルです。

## この不具合について

Windows 11 の一部環境において、アプリの権限アクセス履歴を記録するシステムファイル

```
C:\ProgramData\Microsoft\Windows\CapabilityAccessManager\CapabilityAccessManager.db-wal
```

が異常に肥大化し、ディスク容量を数GB〜数十GB単位で圧迫する不具合が確認されています。

修正更新プログラム(KB5101650)の適用により**新たな肥大化は抑止**されますが、
**すでに肥大化してしまったファイルは自動では縮小されません**。
解消するには「サービス停止 → ファイル削除 → サービス再開」という手動操作が必要で、
コマンド操作に不慣れな方にはハードルの高い作業です。

本ツールはこの一連の操作を自動化したものです。

## ダウンロード

**[Releases ページ](../../releases/latest)** から `CAM_Cleanup.bat` をダウンロードしてください。

ブラウザで完結する配布ページ(解説・ダウンロードボタン付き)も用意しています:
**https://＜ユーザー名＞.github.io/＜リポジトリ名＞/**

## 使い方

1. `CAM_Cleanup.bat` をダウンロードする
2. ファイルをダブルクリックして実行する
3. 「ユーザー アカウント制御」の画面で「はい」を押す

あとは全自動です。処理前後のファイルサイズが画面に表示され、
「処理が完了しました」と出たら終了です。

### 「Windows によって PC が保護されました」と表示された場合

電子署名のない配布ファイル全般に表示される SmartScreen の標準的な確認画面です
(ウイルス検出ではありません)。

1. 画面左上の「**詳細情報**」をクリック
2. 右下に現れる「**実行**」をクリック

## このツールが行うこと

1. 管理者権限への自動昇格(UAC で「はい」を押すだけ)
2. 処理前のファイルサイズを表示
3. Capability Access Manager サービス(`camsvc`)を停止(完全停止まで最大30秒待機)
4. 以下の3ファイルを削除
   - `CapabilityAccessManager.db`
   - `CapabilityAccessManager.db-wal`
   - `CapabilityAccessManager.db-shm`
5. サービスを再開し、処理後のファイルサイズを表示

削除されたファイルは Windows によって正常な状態で自動的に再作成されます。

## 安全性

- 削除されるのは「設定 → プライバシーとセキュリティ」に表示される
  **アプリの権限アクセス履歴(カメラ・マイクの最終使用日時など)のみ**です
- 個人ファイル・アプリ・システム設定には一切影響しません
- 履歴は今後の使用に応じて再び記録されていきます
- サービス停止に失敗した場合やファイルが使用中の場合は、
  **システムに手を加えずに中断**し、再起動後の再実行を案内します

## 動作環境

- Windows 11(Windows 10 でも同名ファイルが肥大化している場合は使用可能です)
- 管理者権限のあるアカウント

## ソースコードの確認

バッチファイルはテキスト形式です。実行前にメモ帳等で開き、
内容を確認したうえでご利用いただけます。
