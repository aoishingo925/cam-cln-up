Windows 11 不具合対策ツール

CapabilityAccessManager
スリム化ツール

Windows 11 の一部環境で、CapabilityAccessManager.db-wal というシステムファイルが 異常に肥大化し、Cドライブの空き容量を数GB〜数十GB単位で圧迫する不具合が確認されています。 修正更新プログラム(KB5101650)の適用で新たな肥大化は抑えられますが、 すでに肥大化したファイルは自動では小さくなりません。
このツールは、その削除作業(サービス停止 → ファイル削除 → サービス再開)を ダブルクリック一発で安全に行うバッチファイルです。

ファイル名: CAM_Cleanup.bat
種類: Windows バッチファイル / 対応: Windows 11

肥大化したファイル(例: 数十GB)を削除し、Windows が数KBの正常なファイルを自動で作り直します。

使い方は3ステップ

  1. 上のボタンからファイルをダウンロード 「CAM_Cleanup.bat」がダウンロードフォルダーに保存されます。
  2. ファイルをダブルクリックして実行 青い警告画面が出た場合は、下の「警告画面が出たとき」の手順で進めてください。
  3. 「ユーザー アカウント制御」の画面で「はい」を押す あとは全自動です。黒い画面に「処理が完了しました」と表示されたら終了です。

警告画面が出たときは(正常な動作です)

「Windows によって PC が保護されました」と表示されたら

この青い画面は SmartScreen という機能によるもので、電子署名のない配布ファイル全般に表示される標準的な確認画面です。 ウイルス検出ではありません。次の2ステップで実行できます。

Windows によって PC が保護されました

Microsoft Defender SmartScreen は認識されないアプリの起動を停止しました。このアプリを実行すると、PC が危険にさらされる可能性があります。

このツールが行うこと

  • 管理者権限への昇格を自動で行います(「はい」を押すだけ)
  • 処理前後のファイルサイズを画面に表示します
  • Capability Access Manager サービス(camsvc)を一時停止します
  • 肥大化したデータベースファイル3点(.db / .db-wal / .db-shm)を削除します
  • サービスを自動で再開します。削除したファイルは Windows が正常な状態で作り直します

安全性について

削除されるのは「設定 → プライバシーとセキュリティ」に表示される アプリの権限アクセス履歴(カメラやマイクの最終使用日時など)だけです。 個人ファイル・アプリ・設定には一切影響せず、履歴は今後の使用に応じて再び記録されていきます。

サービスの停止に失敗した場合やファイルが使用中の場合は、システムに手を加えずに中断し、 「PCを再起動してから再実行してください」と案内する安全設計になっています。
